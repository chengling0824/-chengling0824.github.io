一起学习TypeScript

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript。

TypeScript 是一种由微软开发的自由和开源的编程语言。它是 JavaScript 的一个超集，而且本质上向这个语言添加了可选的静态类型和基于类的面向对象编程。
